#Prueba de Desempeño – Módulo 1 Python

***
A system designed for use in a bookstore,
it allows you to register sales and products, as well as generate detailed reports, apply
discounts by customer and units sold, group statistics by author, and evaluate inventory performance
based on sales.
***

##functionalities
-Show total sales by author
-display the profits with or without discount from an inventory system
-managing a library-oriented inventory
-add items to inventory
-remove items from inventory
-update inventory items

##elements needed to run the program
-Python 3.12.3
-Visual studio code
-Datetime library

##how to run the program
-have the document, the Menu.py file, and Modulo_servicio.py in the same folder.
-from the terminal to the folder
-Type the following in the terminal: python3 Menu.py
And that's it.
